Detailed Description

The MainWindow class is used to hold the GUI elements of the project.

It displays the image, prediction, and label values. It also allows the user to scroll through images and begin training.

Called when an update has been received by the training network.

This will update or create the progress bar GUI with the latest info. When it is shown that the training is complete (batch == total_batches && epoch == total_epochs), the window will close.

Parameters
batchThe current batch number within this epoch.
total_batchesThe total batches for a given epoch.
epochThe current epoch.
total_epochsThe total epochs for which this will run.
latest_lossThe latest loss value from the network.
