void TestMainWindow::updateImage_data ( )
privateslot
Test:
This tests that when an example is received on MainWindow::displayExample, all the appropriate labels are updated.

This includes the:

  • Image label
  • Truth label
  • Prediction label
  • Index label

The test sends two images to make sure that all the values update each time, particularly the image, since it will implicitly share its data.

This uses the logic in updateImage.
