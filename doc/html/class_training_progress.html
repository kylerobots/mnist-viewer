Detailed Description

The TrainingProgress class is used to update the user on the status of training.

It contains two progress bars, one for batch and one for epoch, as well as a label to indicate loss. The progress bars are automatically scaled between 0% and 100% based on the max values provided at construction and the values provided at each update.

It is left to the containing class to determine when to hide this display.
